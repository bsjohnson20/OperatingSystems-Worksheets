
# Worksheet 1
This is an example ReadMe with light selections.  



## Run Locally  

Clone the project  

~~~bash  
  git clone https://gitlab.uwe.ac.uk/
~~~

Go to the project directory  

~~~bash  
  cd OS-worksheet1
~~~


### Compile tasks

**All**
```bash  
make all
```
**One**
```bash
make task{x}
```

**Run tasks**

(1 = compile + run, 0 = compile only)
```bash
./compiling <taskNo> 1
```
